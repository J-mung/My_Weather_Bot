### 목적

사용자에게 현재 위치 및 관심 지역의 날씨 정보를 제공하는 애플리케이션을 개발한다.

본 서비스는 웹 환경과 모바일 뷰 환경을 모두 고려하며,
모바일 환경에서는 빠른 정보 확인과 최소한의 화면 전환을 목표로 한다.

### 주요기능

1. 날씨 정보 조회

- 현재 위치 기반 날씨 조회
- 특정 지역 검색을 통한 날씨 조회

2. 날씨 데이터 표시

- 현재 기온
- 당일 최고 / 최저 기온
- 시간대별 기온 (1시간 간격, 최대 24시간)

3. 사용자 관리 기능

- 검색 지역 즐겨찾기 추가 및 제거

### 유저 액션

1. 사용자는 앱 실행 시 현재 위치의 날씨를 확인할 수 있다.
2. 사용자는 원하는 장소를 검색하여 날씨를 조회할 수 있다.
3. 사용자는 검색 결과를 즐겨찾기에 추가하거나 삭제할 수 있다.
4. 즐겨찾기에서 특정 장소를 선택하면 메인 화면으로 이동하며 해당 지역의 날씨를 표시한다.

### 외부 API

날씨 데이터는 공공데이터포털에서 제공하는 기상청 단기예보 조회 서비스를 활용한다.

본 서비스는 다음 데이터를 제공할 수 있어야 한다.

- 현재 기온
- 시간별 기온
- 일 최저 / 최고 기온
