### 화면별 요소

각 화면에 어떤 요소가 있는지 확정

1. 메인 화면(Main)

- 검색 input-box
- 북마크 화면 이동 버튼
- 현재 날씨 카드
  - 장소명
  - 현재 기온
  - 당일 최고/최저 기온
- 시간대별 날씨 정보
- 북마크 목록

2. 검색 화면(Search)

- 검색 input-box
- 메인 화면 이동 버튼
- 검색 결과 리스트
- 북마크 등록 버튼

3. 북마크(Bookmark) 조회

- 메인 화면, 검색 화면 이동 버튼
- 북마크 목록 편집 버튼
- 북마크 카드 리스트
  - 날씨 Icon
  - 현재 온도
  - 장소명(별칭 등록 가능)
  - 당일 최고/최저 기온

4. 북마크(Bookmark) 편집

- 메인 화면 이동 버튼
- 편집 취소 버튼
- 북마크 카드 리스트
  - 체크박스
  - ...기본항목
- 삭제 버튼
